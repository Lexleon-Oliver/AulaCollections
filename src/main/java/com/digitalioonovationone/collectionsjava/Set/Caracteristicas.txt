#Caracteristicas:

*   Não permite elementos duplicados;
*   Não possui índice;


    *   HashSet
    Não mantem ordem de Inserção.
    Melhor performance
    Permite apenas um elemento nulo

    *   LinkedHashSet
    Mantem ordem de Inserção
    Performance mediana
    Permite apenas um elemento nulo

    *   TreeSet:
    Mantem a ordem natural
    Pior performance
    Não Permite elemento nulo
